Adds a gesture with a specific gestures group to the gestures engine.

Parameters
gestureThe gesture to add to the gestures engine.
groupThe group where the gesture should be added in the gestures engine.
priorityInGroupThe priority of the gesture in the gestures group.
affectedItemsThe items which are affected by the execution of the gesture. Any string is valid as an item key. This prevents gestures from intervering with each other which would most likely lead to conflicting behaviour.
preventConflictsSpecifies whether the gestures engine should prevent the execution of the gesture if a gesture with the same affected items has already been executed before.

Defines and adds a gesture group to the gestures engine.

Parameters
nameThe name of the gestures group.
priorityThe priority of the gestures group. If the priority is the same as an existing one the new gestures group will be added before the existing one.
excludedGroupsThe groups which should be checked before executing this gestures group. Therefore, if an excluded group has already been executed before, this gesture will not be executed.
Returns
The gestures group which was added to the gestures engine.
